# Demo Script: Jal-Kavach

Follow this 60–90 second script to highlight the core capabilities of the app during evaluation.

## Walkthrough Steps

1. **Phase 1: Real-time Awareness (0-20s)**
   - Open the web application. Note the premium dark UI and live location tracking.
   - Click the "Location MapPin" GPS button to fetch your location, or type "Mumbai" and search.
   - Observe current weather conditions, storm details, and any active warnings appearing in the "Active Severe Warnings" panel.

2. **Phase 2: GenAI Preparedness Planner (20-40s)**
   - Click the **GenAI Planner** tab.
   - Choose family size as "4 People" and housing type as "Apartment".
   - Click "Generate Preparedness Plan".
   - Watch the GenAI loader activate. Review the customized Markdown checklist, highlighting structural hazards for apartments.
   - Click **Import Check Items** to sync the generated action items.

3. **Phase 3: Interactive Checklist (40-50s)**
   - The app redirects you to the **Preparedness List** tab.
   - Observe the imported items. Add a custom task like "Buy emergency lights".
   - Check off a few items; notice the progress bar fill up dynamically with glowing teal status!

4. **Phase 4: Route Assessor (50-70s)**
   - Navigate to the **Route Assessor** tab.
   - Type origin "Mumbai" and destination "Lonavala".
   - Click "Analyze Travel Safety".
   - Review the Google Maps driving route render and read the AI Travel Advisory warning about landslide risks on the ghats.

5. **Phase 5: Safety Assistant (70-90s)**
   - Select the **Safety Chatbot** tab.
   - Ask: "What should I do if my car stalls in deep water?"
   - Read the immediate, localized GenAI instructions returned. Try asking in Hindi or another language to see the multilingual translator.
